# UNCLS

Static website for the UNCLS draft framework, built as a multi-page GitHub Pages friendly site. The website can be viewed [here](https://acdu01.github.io/UNCLS/).

Pages:

- `index.html`: Overview of the UNCLS framework and key policy areas.
- `mission.html`: Mission and focus on vintage clothing curators and sellers.
- `financial-support.html`: Grants, funding access, and tax relief for qualified VCCs.
- `grant-application.html`: Grant application checklist with required inputs and storeowner materials.
- `policy-information.html`: Core policy levers, framework structure, and annual session details.
- `contact.html`: Contact channels for draft package questions and session planning.
